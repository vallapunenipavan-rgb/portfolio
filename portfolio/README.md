## 🚀 How to Use

### Option 1: Direct File Opening
1. Navigate to the `portfolio` folder
2. Right-click on `index.html`
3. Select "Open with" → Your preferred browser
4. Or drag `index.html` into your browser

### Option 2: Using a Local Server (Recommended)

**Using Python 3:**
```bash
cd portfolio
python -m http.server 8000
```
Then visit: `http://localhost:8000`

**Using Python 2:**
```bash
cd portfolio
python -m SimpleHTTPServer 8000
```

**Using Node.js (http-server):**
```bash
npm install -g http-server
cd portfolio
http-server
```

**Using Live Server in VS Code:**
1. Install the "Live Server" extension
2. Right-click on `index.html`
3. Select "Open with Live Server"

## 🎨 Customization

### Update Personal Information
Edit `index.html` and replace:
- Contact email
- Phone number
- Location
- GitHub/LinkedIn URLs
- Project descriptions
- Certifications

### Change Color Scheme
In `styles.css`, modify the CSS variables:
```css
:root {
    --primary-color: #0066ff;      /* Change to your color */
    --secondary-color: #00d4ff;    /* Change to your color */
    --dark-color: #1a1a2e;
    --light-color: #f5f7fa;
}
```

### Add New Projects
1. Copy a project card in the projects section
2. Update the title, description, tech stack, and achievements
3. Save and refresh browser

### Add New Skills
1. Find the skills section in `index.html`
2. Add new `<div class="skill-card">` with your skills
3. Add skill tags within it
